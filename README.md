# Blackjack Game in C#

This project is a console-based Blackjack game written in C#.  
It simulates a simplified casino Blackjack table where the player plays against the dealer.

## Features

- Standard 52‑card deck with shuffling
- Player and dealer hands with hit/stand options
- Automatic calculation of hand values (including Aces as 1 or 11)
- Basic betting system and win/lose detection
- Input validation and clear text-based interface

## Purpose

The project was created as a learning exercise to practice:
- C# syntax and basic OOP concepts
- Game state management and control flow
- Working with collections and randomization
- Writing technical documentation (20+ page report describing requirements, design, implementation and testing)

## How to run

1. Open the project in Visual Studio / Rider or another C# IDE.
2. Build the solution.
3. Run the application and follow the instructions in the console.
